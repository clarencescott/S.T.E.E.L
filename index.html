What is S.T.E.E.L?

S.T.E.E.L stands for Students Thriving in Engineering & Emerging Leadership. It is a project offering very affordable pricing to assist high schoolers, graduates, and college students with minimal coding abilities looking to start a web based portfolio, stamping their digital footprint in the changing scope of society and careers.

S.T.E.E.L Portfolio Project

Overview

Welcome to the Engineering Portfolio Project! This initiative is designed to assist students aged 10-17 (and adults 18+) who are passionate about engineering. Participants will build professional web portfolios using HTML, CSS, and JavaScript (including React and Node.js) to showcase their skills, projects, and achievements. The coding/programming langues listed are industry used languages that will equip participants with web development tools, skills and knowledge boosting their credibility.

Features

  • Comprehensive Portfolio Building: Create dynamic and professional web portfolios.
  • Affordable Participation: Only $15 per participant.
  • GitHub Hosting: Portfolios hosted on GitHub Pages (github.io) or Vercel (vercel.app).
  • Long-Term Support: Weekly support until career or college completion.
  • Career & Academic Guidance: Effective showcasing of journey, skills, and achievements.

What's Included

  1. Initial Setup:
    • GitHub account creation
    • Introduction to GitHub Pages
    • Basic setup of HTML, CSS, and JavaScript files
  2. Portfolio Development:
    • HTML: Structure and content
    • CSS: Styling and layout
    • JavaScript: Interactive elements
    • React: Advanced front-end development
    • Node.js: Back-end development
  3. Weekly Support:
    • Video chat sessions via Zoom, Google Video, or in person
    • Code reviews
    • Debugging assistance
    • Feature implementation guidance
    • Best practices in web development
  4. Long-Term Benefits:
    • Continuous portfolio improvement
    • Updates with new projects and skills
    • Preparation for job applications and college admissions

Cost

  • Participation Fee:
    • $15 per student
    • $25 for ages 21+

How to Join

  • Sign Up: Registration Form
  • Payment: Secure your spot by paying the participation fee after the onboarding meeting.
  • Start Building: Begin your journey to create an impressive web portfolio.
  • Github Account: S.T.E.E.L Project Github Repo
  • FaceBook Account: S.T.E.E.L FaceBook Page

Additional Details

  • Prerequisites: Students must fill out a form and submit payment. If under 18, a parent must contact to enroll the child, and adult supervision is preferred.
  • Collaboration Opportunities: Collaborate during weekly coding sessions or on Discord.
  • Progress Tracking: Tracked through GitHub, showcasing coding and programming skills over time.
  • Project Encouragement: Include projects relevant to your career path.
  • Resources Provided: Tutorials and documentation available through YouTube, Google Docs, and Discord.
  • Refund Policy: No refund policy due to the resources provided. The $20 fee grants lifetime access to support and resources.

FAQs

Q: What if I don't have any prior coding experience? A: No problem! This project is designed for all skill levels, and we provide all the necessary resources and support to help you succeed.

Q: How often will I receive support? A: You will receive weekly support sessions to assist with your portfolio development and any questions you might have.

Q: Can I continue updating my portfolio after the project ends? A: Absolutely! Your GitHub repository and GitHub Pages site will remain active, allowing you to continue updating your portfolio as you grow your skills.

Q: Are there any success stories or testimonials from previous participants? A: As this is the first session of this program, we do not have testimonials specific to it. However, we have success stories from individuals we have helped with STEM/STEAM technologies and programming.
